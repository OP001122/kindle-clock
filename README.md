# kindle-clock
个人使用仓库
一个专为 Kindle 浏览器设计的全屏时钟网页应用，支持横屏显示、名言展示和黑白切换。

## 功能特性

- **全屏时钟显示**：大字体时间显示，适合远距离观看
- **日期与星期**：实时显示当前日期和星期
- **名言警句**：每小时自动切换一条名言，点击可手动切换
- **黑底白字模式**：点击"切换黑底白字"可切换配色方案，适合夜间使用
- **横屏适配**：页面自动旋转 90 度，适合 Kindle 横屏浏览
- **UTC+8 时区**：自动适配北京时间
- **自动刷新**：每分钟自动刷新一次，确保时间准确

## Kindle 使用方法

### 基础使用
1.通过usb把静态时钟页面clock.html 文件拷贝到kindle （根目录下）
2.在kindle浏览器输入：file:///mnt/us/clock.html 即可打开网页（无需联网）
3. 在 Kindle 浏览器中打开此网页
4. 浏览器会自动全屏显示时钟
5. 时间每分钟更新一次，名言每小时更新一次

### 关闭熄屏（重要）

Kindle 浏览器默认会定时熄屏，为了让时钟持续显示，请：

1. 在kindle搜索栏的地址栏中输入 `~ds` 并按回车
2. 这会关闭屏幕自动休眠功能，让时钟保持常亮

> **注意**：关闭熄屏后，Kindle 会持续耗电，建议连接电源使用。

### 手动操作

- **切换名言**：点击名言文字区域即可切换下一条
- **切换配色**：点击底部"切换黑底白字"可在黑白配色间切换

## 本地使用

直接用浏览器打开 `clock.html` 文件即可使用。

## 技术说明

- 使用纯 HTML/CSS/JavaScript 编写，无需依赖
- 响应式设计，适配各种屏幕尺寸
- 名言数据内置在脚本中，包含中西方经典名言共 80+ 条
